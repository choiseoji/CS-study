### 📍 KEEP :

1. **[깊이 있는 학습]** : 단순히 답변만 찾는 것이 아니라, 관련 개념이나 꼬리 질문 형태의 지식까지 추가로 서칭하여 정리하면서 더 깊이 있게 이해하려고 노력했습니다.
2. **[복습 습관]** : 스터디 목적으로 만든 자료를 제 블로그 글에 다시 보충하면서 자연스럽게 복습으로 이어질 수 있어서 좋았습니다.

### 💥 PROBLEM :

1. **[전달력 부족]** : 조리 있게 말하지 못해 종종 답변의 요점이 흐려지는 경우가 있었던 것 같습니다.
2. **[시간 관리]** : 발표 시간이 예상보다 초과되는 경우가 있어서 시간 분배에 대한 고민이 필요하다고 느꼈습니다.
3. **[범위 설정]** : 준비하는 내용이 스터디 범위 밖으로 벗어나는 것은 아닌지 다시 한번 점검할 필요가 있다고 생각했습니다.

### 👍 TRY :

1. **[두괄식 답변 연습]** : 발표나 질의응답 시간에 결론부터 말하고 뒤에 상세 설명을 붙이는 두괄식 답변을 연습해보려 합니다.
2. **[자기 점검 습관]** : 발표하고 끝내는 것이 아니라 제가 진짜로 다 이해했는지, 자료를 보지 않고도 설명할 수 있는지 스스로에게 질문해보는 습관을 들이고 싶습니다.
